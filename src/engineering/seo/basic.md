# SEO 基础

SEO (搜索引擎优化) 是通过改进网站内容和结构，提升在搜索引擎自然搜索结果中排名的过程。对于前端开发而言，SEO 关注的是如何通过技术手段让网站更容易被搜索引擎理解和收录，从而获得更多免费流量。

## 什么是 SEO？

SEO 是让网站在搜索引擎中获得更高排名的系统性工作，分为站内优化和站外优化。

```
搜索引擎工作流程:
[爬虫抓取] -> [内容解析] -> [索引存储] -> [排名计算] -> [结果展示]
      |           |           |           |           |
  发现页面     分析内容     建立索引     算法评分     用户看到
```

## HTML 标签优化

### 标题标签

使用层次分明的标题结构。

```
标题层级:
h1: [页面主标题] - 唯一且包含核心关键词
h2: [章节标题] - 支持主标题的内容块
h3: [子章节] - 进一步细分内容
```

### 元标签

提供页面摘要信息给搜索引擎。

```
元标签组成:
<title>页面标题 - 品牌名</title>
<meta name="description" content="页面描述">
<meta name="keywords" content="关键词1,关键词2">
```

### 语义化 HTML

使用有意义的 HTML 标签。

```
语义化结构:
<header>    页面头部
<nav>       导航菜单
<main>      主要内容
<article>   独立内容块
<section>   内容章节
<aside>     侧边栏
<footer>    页面底部
```
